📄 llms.txt: AI 로봇만을 위한 전용 안내 지도

홈페이지에 AI 전용 초대장을 심습니다.
ChatGPT나 Perplexity가 우리 병원의 핵심 정보(원장님 약력, 특화 진료)를
우선적으로 학습하여 환각 없이 정확하게 답변하도록 만듭니다.

  • AI가 정확한 정보를 답변하도록 유도
  • 경쟁 병원보다 먼저 선점
  • 글로벌 AI 표준 규격

# 서울더좋은치과 핵심 정보

> 임플란트 전문, 강남역 3분, 야간진료

## 의료진
- 김서울 대표원장: 보건복지부 인증 전문의
  임플란트 15년 경력, 오스템 임플란트 자문위원

## 진료 철학
- 통증 최소화: 무통마취기 보유
- 과잉진료 없는: 양심치과 리스트 등재

🏷️ FAQ 스키마: 질문/답변 꼬리표 달기

홈페이지 질문/답변 섹션에 컴퓨터 언어(JSON-LD)로 꼬리표를 달아줍니다.
AI 검색 엔진이 "이것이 질문이고, 이것이 정답이구나"라고
100% 이해하게 만들어, 추천 답변으로 뜰 확률을 30% 이상 높입니다.

  • 구글/네이버 검색 결과 상위 노출
  • AI가 신뢰하는 데이터 구조
  • 음성 검색(Siri, Bixby) 최적화
JSON-LD (FAQ 스키마) 
/* AI가 이해하는 질문/답변 코드 */
{
  "@type": "FAQPage",
  "mainEntity": [{
    "@type": "Question",
    "name": "임플란트 비용은 얼마인가요?",
    "acceptedAnswer": {
      "@type": "Answer",
      "text": "오스템 정품 기준 80만원부터..."
    }
  }]
}

📋 실전 사례

SEO에서 AEO 리빌딩

원장님 블로그 글을 AI가 좋아하는 형식으로 변환한 시뮬레이션입니다.
이 차이가 'AI의 선택'을 결정합니다.

1 구조 혁신: 두괄식 답변 (Inverse Pyramid)

AI는 서론이 긴 글을 싫어합니다. 첫 50자 내에 직접적인 답변을 배치해야 합니다.

❌ Before (기존 블로그) ✅ After (AEO 최적화)
"안녕하세요, 마곡 수면치과입니다.
요즘 날씨가 많이 추워졌는데요,
오늘은 많은 분들이 궁금해하시는
수면 임플란트에 대해..."

→ 결론(가능 여부)이 글 중반부에 등장 		"수면 임플란트는 가능합니다.
의식하진정법(레미마졸람)을 사용하면
통증 없이 30분~1시간 내 시술이
완료됩니다."

→ 첫 문장에 답변 완료
💡 핵심: AI는 첫 50자를 가장 중요하게 읽습니다. 인사말은 생략하고 바로 답변하세요.

2 데이터 구조화: 줄글 대신 '표(Table)' 활용

AI는 텍스트 나열보다 구조화된 표나 리스트를 훨씬 더 잘 이해하고 인용합니다.

❌ Before (줄글 설명) ✅ After (표 형식)
"전신마취는 완전히 의식을 잃는
방법이고요, 의식하진정법은
반수면 상태로 진행됩니다.
전신마취는 마취과 전문의가
필요하고 비용도 더 비쌉니다..."
전신마취 의식하진정
의식 완전 소실 반수면
회복 2~3시간 30분
비용 50~80만 15~25만
💡 핵심: AI는 표 형식 데이터를 그대로 인용합니다. 비교 정보는 반드시 표로 정리하세요.

3 신뢰도 강화: '팩트 밀도(Fact-Density)' 높이기

모호한 표현 대신 구체적인 수치와 통계를 제시해야 AI가 신뢰도 높은 정보(E-E-A-T)로 인식합니다.

❌ Before (모호한 표현) ✅ After (구체적 수치)
"회복이 빠르고 안전합니다."

"통증이 거의 없습니다."

"많은 분들이 만족하셨습니다." 		"레미마졸람 사용 시
평균 30분 이내에 의식이 회복되며,
시술 중 산소포화도(SpO2)
95% 이상을 유지합니다."

→ 숫자와 의학 용어로 신뢰도 UP
💡 핵심: "빠름" → "30분", "안전함" → "SpO2 95%" 처럼 수치로 바꾸면 AI 신뢰도가 올라갑니다.
